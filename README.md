## 📌 Overview

**PT1** and **Codesoft** are structured Python practice repositories built during the foundational phase of engineering. Together they cover Python fundamentals, algorithmic thinking, reusable scripting patterns, and real-world problem solving — forming the bedrock on which larger projects like AQUA, Jarvis, and Here-and-Now-AI were built.

---

## 📁 Repository Breakdown

### 🧠 [PT1 — Python Foundations](https://github.com/Bhogi123-T/PT1)

Core Python concepts, language features, and reusable utility scripts.

| Category | Topics Covered |
|----------|---------------|
| 🧩 Core Python | Data types · Control flow · Functions · OOP |
| 🛠️ Utilities | File I/O · String manipulation · Data structures |
| 📐 Logic Building | Recursion · Iteration · Pattern logic |
| ✅ Clean Code | Readable, well-structured, commented scripts |

### 💻 [Codesoft — Programming Practice](https://github.com/Bhogi123-T/codesoft)

Algorithmic practice, logic building, and structured problem-solving.

| Category | Topics Covered |
|----------|---------------|
| 🧠 Algorithmic Thinking | Sorting · Searching · Greedy algorithms |
| 💡 Problem Solving | Practice problems with structured solutions |
| 🧩 Logic Patterns | Mathematical programming · Pattern recognition |
| 🏗️ Project-Oriented | Mini-projects applying Python fundamentals |
